# WeatherDBd3
## Web component

## Libraries Used

* d3.js (http://d3js.org/)
* NVD3 (http://nvd3.org/)
* BootStrap (http://getbootstrap.com/)

## Develop Locally

`$ git clone https://github.com/iann/iann.github.com.git`

`$ cd projects/weather`

`$ python -m SimpleHTTPServer`

Add data to `data` directory and reference it in html. Use either d3 or NVD3 supported formats.
