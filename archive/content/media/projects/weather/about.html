WeatherDBd3

The National Oceanic and Atmospheric Administration (NOAA) in the United States and other international atmospheric agencies have tens of thousands of meteorological measurement collection stations across the globe. These stations make continuous measurements of data points such as -- air temperature, dew point temperature, sea level pressure, wind direction, wind speed rate, cloud coverage, one hour precipitation total, and six hour precipitation total.

The Integrated Surface Database is a collection of hourly readings from these stations. Currently, there are about 20,000 active stations reporting with some of the first readings originating from 1901.

The ISD is readily available to anyone with an internet connection through an FTP server. However, when one navigates to this page they see what appears to be unorganized compressed text files. This is not a user-friendly format and inhibits the users ability to actually see the data these files contain.

Since this is intended to be a web application anyone in the world can view and manipulate the data. WeatherDBd3 aims to give a casual user with only a web browser perspective into this interesting dataset being collected all around them.
